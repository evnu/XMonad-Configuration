XMonad Configuration
====================

## Description
The files in this repository are my configuration files for XMonad.


## Requirements
* XMonad (>= 0.9.1)
* XMonad Contrib (>= 0.9.1)
* aumix-gtk (>= 2.9.1-1?)
* rxvt-unicode
* osdbattery (>= 1.4-3)

## Installation
Link to `~/.xmonad` and provide another link to xmobarrc.
